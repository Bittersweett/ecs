# 查询弹性云服务器单个磁盘信息<a name="ZH-CN_TOPIC_0167811961"></a>

## 功能介绍<a name="zh-cn_topic_0101860614_section61843920"></a>

该接口支持企业项目细粒度权限的校验，具体细粒度请参见   [ecs:cloudServers:get](磁盘管理（API授权）.md#li1650621815611)
